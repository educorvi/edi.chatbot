## Project setup to use demo Application
#### Install Dependencies
```
npm run install
```
#### Run Tests
```
npm test
```
#### Compiles and hot-reloads for development
```
npm run serve
```
#### Compiles and minifies for production
```
npm run build
```

## Also usable as npm package
```
npm install edi.chatbot
```
